# Cloudflare 支持 Vite 的使命 {#cloudflare-supports-vites-mission}

_2026年6月4日_

![Cloudflare 支持 Vite 使命的封面图](/og-image-cloudflare-supports-vite.webp)

今天标志着 Vite 新阶段的开始：VoidZero 将加入 Cloudflare。完整故事可阅读 Evan You 撰写的 [VoidZero 公告](https://voidzero.dev/posts/voidzero-cloudflare)。[Cloudflare 的博客文章](https://blog.cloudflare.com/voidzero-joins-cloudflare/) 解释了他们为何投资 Vite 技术栈。如今，Vite 技术栈已经成为 Web 基础设施的一部分。Vite 的团队治理、使命与理念不会改变：

- Vite 继续以 MIT 许可证开源。
- Vite 继续保持厂商中立，使用 Vite 构建的应用可以在任何平台运行。
- Vite 继续由 [Vite 团队](/team) 负责维护。团队既有受雇于不同组织的成员，也有独立成员。
- [Vite Open Collective](https://opencollective.com/vite) 的资金由 Vite 团队管理，用于资助独立核心团队成员、依赖项目以及生态系统中的其他关键项目。
- 所有受雇于 VoidZero 的 Vite 团队成员都将加入 Cloudflare，并继续参与 Vite 的工作。

如 VoidZero 和 Cloudflare 的公告所述，[Vitest](https://vitest.dev/team.html)、[Rolldown](https://rolldown.rs/team)、[Oxc](https://oxc.rs/team.html) 和 [Vite+](https://viteplus.dev/team) 同样如此。

## Cloudflare 的 Vite 生态系统开源基金 {#cloudflares-vite-ecosystem-open-source-fund}

Cloudflare 致力于帮助 Vite 保持中立，并宣布为 Vite 生态系统设立一项新的 100 万美元开源基金。结合继续由 Vite 团队管理的 Vite Open Collective，这些资金将帮助我们进一步改进 Vite 及其生态系统，包括：

- 资助生态系统中更多广受欢迎的插件和工具。
- 为核心团队成员提供津贴，以支持独立成员。
- 与 Rolldown、Oxc 和 Vite+ 等 Vite 技术栈团队密切合作。
- 与框架、插件和部署平台的生态系统维护者密切合作。
- 与标准组织合作，更快采用新的 Web 平台功能。
- 更快地审计并发布安全问题。

其中许多工作我们已经着手开展，这笔基金将帮助加快推进进程。

## 感谢 {#thank-you}

我们要感谢目前通过 [GitHub Sponsors](https://github.com/sponsors/vitejs) 和 [Open Collective](https://opencollective.com/vite) 支持我们的赞助者，你们的支持对我们依然弥足珍贵。我们也要感谢各个项目、生态系统框架和插件团队，正是彼此的合作帮助 Vite 走到今天。同时还要感谢 Vite 仓库中 [1250 多位贡献者](https://github.com/vitejs/vite/graphs/contributors)，以及持续提供宝贵反馈、帮助 Vite 改进的用户。

我们在 [Vite 8 发布公告](/blog/announcing-vite8#looking-ahead) 中讨论了下一步计划。全量打包模式已经取得良好进展；新设立的生态系统同步会议汇集了不同框架、部署平台和插件的成员，成为催生新想法、进一步扩展 Vite 能力的重要动力。我们相信，Vite 的环境 API 正沿着在 Vite 9 中实现稳定的良好轨迹前进。随着 Vite 的故事继续展开，我们期待在未来的漫长道路上与大家共同构建 Web。
